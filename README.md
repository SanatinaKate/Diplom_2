## Дипломный проект. Задание 2: API

### Автотесты для тестирования API сервиса Stellar Burgers - https://stellarburgers.nomoreparties.site/

Ссылка на документацию API - https://code.s3.yandex.net/qa-automation-engineer/python-full/diploma/api-documentation.pdf?etag=3403196b527ca03259bfd0cb41163a89

### Реализованные сценарии

Созданы автотесты, покрывающие следующие сценарии:
- Создание пользователя
- Логин пользователя
- Изменение данных пользователя
- Создание заказа
- Получение заказов конкретного пользователя

Применена техника параметризации

Отчет Allure о результатах запуска автотестов: allure-report/index.html

### Структура проекта

Проект выполнен с использованием фреймворка pytest и библиотеки requests

В директории tests находится пакет с автотестами:
- файл test_order_create.py содержит класс `TestOrderCreate` с тестами для сценариев создания заказа
- файл test_orders_list.py содержит класс `TestOrdersList` с тестами для сценариев получения заказов конкретного пользователя
- файл test_user_login.py содержит класс `TestUserLogin` с тестами для сценариев логина пользователя
- файл test_user_register.py содержит класс `TestUserRegister` с тестами для сценариев создания пользователя
- файл test_user_update.py содержит класс `TestUserUpdate` с тестами для сценариев изменения данных пользователя

В директории methods находится пакет с методами API для автотестов:
- файл orders_methods.py содержит класс `OrdersMethods` c методами для работы с заказами
- файл user_methods.py содержит класс `UserMethods` c методами для работы с пользователями

Файл data.py содержит класс `Data` со статическими данными для автотестов

Файл helpers.ru содержим класс `Helpers` со статическими методами генерации случайной строки и случайного пользователя

Файл conftest.py содержит pytest-фикстуры для регистрации/удаления пользователя и создания заказов

Файл requirements.txt содержит версии пакетов, необходимых для работы проекта

### Запуск автотестов

**Установка зависимостей**

> pip install -r requirements.txt

**Запуск автотестов и генерация отчета Allure**

> pytest tests/ --alluredir=allure_results
> allure generate --single-file allure_results
